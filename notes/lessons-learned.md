# Lessons Learned

## What I Learned

This project taught me more than just how to fix a game.

I learned how Proton and Wine work together to run Windows applications on Linux. I also learned how to read GitHub projects, follow technical documentation, and troubleshoot problems by testing different solutions instead of guessing.

Building the patched DLLs helped me understand that open source software can often be fixed by the community before an official update is released.
